# SoundFont directory

The repository does not contain `FluidR3_GM.sf2` because the file is larger than GitHub's normal 100 MB limit.

Place a General MIDI compatible SoundFont here using the default name:

```text
FluidR3_GM.sf2
```

Expected full path:

```text
agent/soundfonts/FluidR3_GM.sf2
```

Alternatively, run the root startup script with another `.sf2` path:

```powershell
.\start.ps1 -SoundFontPath 'D:\SoundFonts\another.sf2'
```
